# 1.2.1 简单的乐谱

LilyPond 会自动添加一些乐谱元素。在下面的例子中，我们仅仅指定了四个音，但 LilyPond 加上了高音谱号，拍号和节奏。

```text
{
    c' e' g' e'
}
```

![](../../../.gitbook/assets/1.2.png)

这些行为也许需要改变，但在大多数情况下，自动生成值是很有帮助的。
